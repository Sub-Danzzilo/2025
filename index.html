Projects

Temperature Converter

Temperature Converter is a website that converts temperatures from Celsius, Fahrenheit, Kelvin, and Reaumur. The output displays not only the results but also an explanation of the calculations behind the conversion from temperature A to temperature B. The Temperature Converter website offers additional features like dark mode and music. The website's UI/UX design is iOS-like, making it minimalist and pleasing to the eye. This website project is intended for the RevoU mini-course.

Heart Sync

Heart Sync is a fun website that shows how much you love someone. Simply enter your name or someone else's name, then the name of the target person. When you press the "Sync Heart" button, a random number generator will appear. Click the heart icon to change the heart color. This project is just for fun and dedicated to my friend.

Space Defender

Space Defender is a space shooting game that only requires WASD or Arrow keys to move, without the need for manual shooting. It has a power-up system to make it easier and leveling. The higher the level, the more challenging the opponents. This game was created for a final semester project in the Algorithms Programming course. It was developed using pure Python with the PyGame library and Pyinstaller to build the game into an executable file. The game can be played remotely in multiplayer mode using ZeroTier.
